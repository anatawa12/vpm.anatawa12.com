    Clear Endpoint Position

    Clear Endpoint Position #

    このコンポーネントは、Endpoint Positionの値を0に置換し、_EndPhysBoneGameObjectを追加します。

    このコンポーネントはPhysBoneコンポーネントがあるGameObjectに追加してください。

    今のところ、このコンポーネントに設定項目はありませんが、ボタンが1つあります。 適用して今すぐcomponentを削除するをクリックすると、 この操作が適用され、コンポーネントが削除されます。 アバターを制作するときに便利かもしれません。
