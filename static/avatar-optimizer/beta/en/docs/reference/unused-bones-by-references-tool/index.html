    UnusedBonesByReferencesTool

    UnusedBonesByReferencesTool #

    Marks it as EditorOnly that bones referenced only from EditorOnly meshes.

    This component should be added to an avatar root. (Kind: Avatar Global Component)

    This component is obsolete. You should use Remove unused Objects of Trace and Optimize instead. This component will never get algorithm improvements.

    This is port of UnusedBonesByReferencesTool by Narazaka but run at build time.
