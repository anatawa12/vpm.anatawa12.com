    Rename BlendShape

    • 設定

    Rename BlendShape #

    メッシュのBlendShape名を変更することが出来ます。
    変更後の名前が重複している場合、それらは1つのBlendShapeへと統合されます。

    このコンポーネントは、SkinnedMeshRendererコンポーネントのあるGameObjectに追加してください。(分類: Modifying Edit Skinned Mesh Component)

    設定 #

    BlendShape名の変更一覧表が表示されます。 右上の ‘+’ ボタンから、名前を変更するBlendShapeを追加することが出来ます。

    変更後の名前が重複している場合は、それらが統合されることを示すために、入力欄が黄色でハイライトされます。
