    Make Children #

    ビルド時に、指定されたGameObjectをこのGameObjectの子にします。

    このコンポーネントは非推奨です。

    この機能はアバターの最適化と関係がないため、今後製作予定の非破壊アバター改変ユーティリティのコンポーネントに置き換えられる予定です。
